# Let's Matching!
🧑‍🤝‍🧑 Firebase의 RealtimeDatabase와 cardStackview를 통한 매칭시스템 🧑‍🤝‍🧑

## 개발환경

* OS : windows 10 pro
* IDE : Android Studio
* Language : Kotlin

## 공부내용

* Firebase Authenetication 사용
  1. email Login
  2. Facebook Login
* Firebase Realtime Database 사용
* yuyakaido/CardStackView 사용
